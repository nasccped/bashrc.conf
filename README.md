bashrc config
=============

Here is my `.bashrc` configurations!

## Set up the configurations

1. Clone the repository
```sh
git clone --depth 1 https://github.com/nasccped/bashrc.conf
```
2. `cd` + remove git folder
```sh
cd bashrc.conf
rm -rf .git
```
3. Add the config
```sh
echo ./conf/.bashrc >> $HOME/.bashrc
```
